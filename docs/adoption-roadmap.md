# Adoption roadmap

## Phase 1: minimum guardrails

Install:
- `AGENTS.md`
- `.codex/config.toml`
- `review-owner`
- `plan-feature`
- `fix-issue`

Outcome:
- better request framing
- smaller diffs
- more honest validation

## Phase 2: architecture gate

Install:
- `project-bootstrap`
- `brownfield-mapping`
- `architecture-discovery`
- `nfr-capture`
- `architecture-review`
- `plans/templates/`

Outcome:
- fewer accidental design mistakes
- better scale and maintainability decisions

## Phase 3: release discipline

Install:
- `migration-guard`
- `observability-review`
- `docs-sync`
- `release-readiness`

Outcome:
- safer rollouts
- fewer operator surprises

## Phase 4: automation

Enable:
- PR review workflow
- architecture gate workflow
- docs drift workflow
- issue triage workflow
- release readiness workflow
