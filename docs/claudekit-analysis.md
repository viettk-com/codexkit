# ClaudeKit analysis that shaped v2 and v3

The uploaded ClaudeKit snapshot is valuable because it treats agentic coding as a governed workflow system, not a prompt toy.

## Patterns worth preserving

- planning before implementation
- specialist agents instead of one giant prompt
- explicit plans, reports, and docs
- strong docs culture
- worktree routines for isolated parallel work

## Patterns intentionally not ported 1:1

- giant command trees
- heavy host-specific hook logic
- provider-specific runtime behavior
- duplicated features Codex already supports natively

## What v2 and v3 take from ClaudeKit

This bundle keeps the strongest high-level ideas:

- durable guidance
- specialist agents
- reusable skills
- explicit artifacts
- review and release rigor

Then it adds a stronger architecture-first lane for greenfield and brownfield work.
