# Project context machine profile

This directory is managed by CodexKit bootstrap.

After running `scripts/bootstrap-codexkit.py --apply`, it contains:

- `profile.json` - machine-readable repository profile
- `module-index.json` - discovered modules and subsystem map
- `dashboard.json` - active initiatives, hotspots, and recent git signals
- `continuity.json` - recurring lessons and active initiative summaries
- `rules.json` - condensed guardrails for agents and automations
